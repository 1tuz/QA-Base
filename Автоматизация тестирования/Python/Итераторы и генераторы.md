‚úÖ **–ü—Ä–∏–º–µ—Ä –∏—Ç–µ—Ä–∞—Ç–æ—Ä–∞**
```python
class MyIterator:
    def __init__(self, limit):
        self.limit = limit
        self.counter = 0

    def __iter__(self):
        return self

    def __next__(self):
        if self.counter < self.limit:
            self.counter += 1
            return self.counter
        raise StopIteration

for i in MyIterator(3):  # 1, 2, 3
    print(i)
```

```python
def my_generator():
    for i in range(3):
        yield i

for i in my_generator():  # 0, 1, 2
    print(i)
```

‚úÖ **–†–∞–∑–Ω–∏—Ü–∞ `return` –∏ `yield`**
- `return` **–∑–∞–≤–µ—Ä—à–∞–µ—Ç** —Ñ—É–Ω–∫—Ü–∏—é.

‚úÖ **–ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ `next()`**
