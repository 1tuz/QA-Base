#### **Синтаксис:**

```python
if условие:
    # код
elif другое_условие:
    # код
else:
    # код
```

#### **Пример:**

```python
x = 10
if x > 0:
    print("Положительное")
elif x == 0:
    print("Ноль")
else:
    print("Отрицательное")
```

#### **Короткая запись:**

```python
result = "Положительное" if x > 0 else "Отрицательное"
```
